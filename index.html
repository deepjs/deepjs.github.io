deepjs core

is a namespace that gathers and links nicely a bunch of atomic and generic powerful libraries.
Each of them could be used independently, or together where they release all their potential.
Really useful and unobstrusive, they could be used anywhere for any kind of project.

 

decompose

function forgery tools
 

promise & context

asynchrone & concurrent programming

 

protocols

abstract resource locators
 

compiler, flatten & classes

inheritance and specialisation
 

nodes, query & traversal

objects properties selection & handeling
 

modes & ocm

manage who, where & how
 
   

deepjs addons

cross modules tools definition



getting started


To install deepjs core, you could use bower or npm to install last version of deepjs.

> npm install --save deepjs

or

> bower install --save deepjs


To get a fully workable example on how it's loaded in a web page, you could clone this site :

> git clone https://gitub.com/deepjs/deepjs.github.io
> npm install
> bower install
> gulp serve
